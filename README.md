# csgo-autoexec

Go to the autoexec location, and run the following command:
```
git clone git@github.com:cyberhck/csgo-autoexec autoexec
echo 'exec "autoexec/autoexec"' > autoexec.cfg
```
Now loading csgo should load this project.
